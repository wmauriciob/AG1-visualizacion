Los ajustes requeridos de tamaños de las fuentes,colores y demás parámetros solicitados. Además, se presemtaron las siguientes iniciativas:
Hoja 1 del informe: se incluyeron etiquetas de datos para las categrorías de los ggéneros y para el total los presupuestos.
Hoja 1 del informe: se incluye la funcionalidad de reescalar el eje horizontal para discriminar valores de presupuesto específico con el fin de identificar características de los distribuidores y el género para valores específicos de presupuesto. Esta estrategia se usó también para las hojas 2 y 3 del informe.
